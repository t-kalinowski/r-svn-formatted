Search Engine

This search engine is in alpha stage ... don't expect too much. Currently you can only search for functions of the base package, but searching all packages should be implemented soon.

Search

You can search for keywords, function and data names and titles.

Usage: Enter a string in the text field below and hit RETURN.

Keywords only

Keywords

Eventually you will be able to dynamically build an alphabetical list of all keywords or look at all keywords ordered by topic. Currently there is only the demo section "Graphics".

Keywords by Topic

Graphics
  • aplot: Add to Existing Plot / internal plot.
  • dplot: Computations Related to Plotting
  • hplot: High-Level Plots
  • iplot: Interacting with Plots
  • color: Color, Palettes etc.
  • dynamic: Dynamic Graphics
  • device: Graphical Devices
